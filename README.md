# Patterna 🧩

[![Deploy to GitHub Pages](https://github.com/patterna-project/patterna-project.github.io/actions/workflows/deploy.yml/badge.svg)](https://github.com/patterna-project/patterna-project.github.io/actions/workflows/deploy.yml)
[![Vercel](https://img.shields.io/badge/Vercel-Live-000?logo=vercel)](https://patterna-project-github-io.vercel.app)

Webová aplikácia pre generovanie optimálnych sekvencií organizačných vzorov pomocou **Markov Decision Process (MDP)** a analýzy textovej podobnosti.

🔗 **Live demo:** [patterna-project-github-io.vercel.app](https://patterna-project-github-io.vercel.app)  
📚 **Bakalársky projekt:** FIIT STU Bratislava, 2025/2026

---

## ✨ Hlavné funkcie

- 🧠 **Generovanie sekvencií** – pomocou MDP a value iteration
- 📊 **Matica podobností** – vizualizácia vzťahov medzi vzormi
- 🕸️ **Interaktívny graf** – prehliadanie súvislostí (D3.js)
- 🤖 **AI hodnotenie** – sekvencie hodnotené umelou inteligenciou (Gemini API)
- 📁 **Vlastné katalógy** – nahrajte vlastné vzory s podpriečinkami
- 🌙 **Dark mode** – podpora svetlého/tmavého režimu
- 🌐 **Dvojjazyčné rozhranie** – slovenčina / angličtina
